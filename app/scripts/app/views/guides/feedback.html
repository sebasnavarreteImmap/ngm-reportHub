Feature Request

Purpose: Have a cool idea for some new features? Send the details to the ReportHub team so they can improve the system for you!

sendSend To: ReportHub Team @ ngmReportHub@gmail.com

Subject: Feature Request

Details: In the email please include the details and a description as listed below;

Field Description
Name The name of the new feature. i.e. "New Dashboard Filter"
Purpose The purpose of the new feature. i.e. "To filter by donor/organization"
Description A description of the new feature. i.e. "Have a filter menu on the dashboard"
Bug Report

Purpose: Found an error or something is not working? Let the ReportHub team know so they can help!

sendSend To: ReportHub Team @ ngmReportHub@gmail.com

Subject: Bug Report

Details: In the email please include the details and a description as listed below;

Field Description
Page The name of the page where you encountered the error. i.e. "New Project Page"
URL The URL of the page where you encountered the error
Action The action you were performing at the time. i.e. "Adding a new Beneficiary"
Error Any error messages that the ReportHub displayed
Description Any additional description of the error that will prove useful. i.e. "I removed a location and then added a new location"
